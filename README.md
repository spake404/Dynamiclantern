# Dynamic Lantern

Dynamic Lantern is a Forge 1.20.1 client-side visual mod for Curios lanterns.

It lets vanilla lanterns equipped in a Curios belt slot render on the player's waist with a small swinging animation. When Iris or Oculus is installed, Curios lanterns are also exposed to shader packs as held-item lighting, so compatible shaders can make them glow.

## Features

- Supports `minecraft:lantern` and `minecraft:soul_lantern` in Curios.
- Renders equipped lanterns on the player's waist.
- Adds configurable swinging physics.
- Supports Iris/Oculus held-item shader lighting for Curios lanterns.
- Includes Epic Fight combat model compatibility.
- Does not add dynamic light sources.

## Requirements

- Minecraft 1.20.1
- Forge 47+
- Curios

## Optional Compatibility

- Iris or Oculus for shader-pack lighting.
- Epic Fight for combat model support.
